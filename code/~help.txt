============ Инструкция по работе с программой VisualXML. ==================

Для того, чтобы загрузить новый XML файл, нажмите кнопку "Load XML File", 
обработанный файл появится в поле для редактирования.

Документ xml обработанный по некоторым правилам, 
каждая строка содержит следующую информацию:
<T>: <тег>:<'атрибут1':'значение атрибут1', 'атрибут2':'значение атрибут2'...'атрибутN':'значение атрибутN'>
где T - глубина дерева на котором находится данный тег после которого следует соответсвующее значению
количество отступов, для лучшей наглядности представления данных.

Поле, находящееся справа в меню, предназначено для выбора определённого тега из списка
тегов, находящихся в файле.
При нажатии на тег в списке, на экране будут отображены только атрибуты данного тега, 
которые программа нашла в исходном XML файле.
Пока не загружен документ, поле тегов будет пустым.

Кнопка "Show file" снова полностью показывает обработанный XML файл, после просмотра отдельных тегов.

Последняя кнопка "Save into file" - сохраняет текст из поля в .txt файл, для дальнейшей работы.
